Автозагрузка

Файлы показателей

Загрузите Excel-файл со структурой месяц / ФИО / ИТОГО. Значение будет записано в показатель Исполнительская дисциплина только у тех сотрудников, у кого этот показатель назначен по должности в выбранном периоде.

Загрузите Excel-файл со структурой месяц / Сотрудник / всего / просрок / исполнение %. Значение из поля исполнение % будет записано в показатели Исполнение сроков согласования договоров или Соблюдение сроков заключения договоров только у тех сотрудников, у кого этот показатель назначен по должности в выбранном периоде.

Загрузите export-выгрузку со строками id / Обект / Руководитель / typ / 01..12. По выбранному KPI-месяцу система возьмет нужную колонку месяца, соберет данные по объектам, усреднит KPI по руководителю и обновит показатели Исполнение Плана поступления ДС, Исполнение Плана расходования ДС и Исполнение Плана выполнения СМР, руб.
